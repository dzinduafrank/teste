Nossa Empresa

O processo inicia com a instalação dos equipamentos nos campos agrícolas, estes equipamentos representam os conjuntos de componentes previstos para análise do solo por meio de sensores adequados a problema em questão. Em seguida o envio os dados a partir do dispositivo ZigBee coordenador que é responsável pela comunicação entre os sensores e a central de processamento, através de uma aplicação são enviados para laboratório onde são comparados com base de dados de todas as mostras disponível e são enviados para ao produtor. Os sensores geram sinais analógicos de corrente que são proporcionais as bactérias presentes e a mineralização de substâncias orgânicas e nutrientes. Os quais são enviados para as entradas analógicas dos módulos de controlo, por meio das linhas de comunicação de sensor que são dedicadas e apropriadas para essa função. As placas fazem a conversão e adequação desses sinais de corrente e os enviam.

Sensores-2

O S05-SM é um sensor de umidade do solo projectado para a captura de dados de precisão na agricultura. Este dispositivo sem fio é alojado em uma caixa IP66 certificada eliminando a necessidade de instalação de fios e dando-lhe uma alta resistência a condições climáticas extremas. O S05-SM é alimentado por bateria e tem uma faixa de transmissão de 500m tornando-o adequado para instalação nos campos. Este dispositivo também é compatível com ZigBee Sentrol Cloud e ZigBee Complete Solution para colecta de dados constantes, reconhecimento instantâneo e controle de ambiente automatizado.

FTM é um sensor que é inserida no solo para estimar a fertilidade de acordo com as progressivas de enfraquecimento e de algodão de seda fibras, devido à actividade microbiana. O sistema usa oito fios, dois seda e algodão dois não tratados (de referência) e 2 outra seda e algodão 2 impregnado com azoto e fosfato de potássio. A relação entre os diferentes tempos de ruptura do fio fornece um conjunto de informação relacionada com o tipo de bactérias presentes e o tempo de mineralização de substâncias orgânicas e, por conseguinte, qualquer deficiência de nutrientes e a actividade do solo
